## DESIGN PATTERNS
##### Proven solution to solve design problems.

> 3 Basic Types: **creational, structural and behavioral**.

### Creational

- Abstract Factory
- Builder
- Factory Method
- Object Pool
- Prototype
- Singleton

### Structural

- Adapter
- Bridge
- Composite
- Decorator
- Facade
- Flyweight
- Proxy

### Behavioral

- Chain of Responsibility
- Command
- Interpreter
- Iterator
- Mediator
- Memento
- Observer
- State
- Strategy
- Template Method
- Visitor

> These 3 groups define how program elements relate to each other, how they get created and how they communicate with each other.
